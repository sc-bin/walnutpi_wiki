核桃派系统支持通过终端命令切换操作系统语言，目前支持中文和英文两种语言。**首次切换需要时间长一点，大约几十秒，请耐心等待。**设置完成后重启核桃派生效：

- 切换为中文：

```bash
sudo set-language cn
```

- 切换为英文：

```bash
sudo set-language en
```
